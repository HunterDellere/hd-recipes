# Cuisine Reference Notes

Internal reference loaded at recipe drafting time. Not built into published pages (the `_reference/` prefix is a build-time exclusion convention shared with `_schema/` and `_featured/`).

## What these are

Each file is a descriptive briefing on a broad cuisine region — what makes each tradition recognizably itself, the spectrum of legitimate variation, the pantry foundations, the defining techniques, and common Western drift to be aware of. Not a checklist; a knowledgeable friend's orientation.

The drafting workflow loads the matching note before drafting any recipe in that region. The goal is informed drafting, not constrained drafting.

## Files

- `east-asian.md` — China, Korea, Japan
- `southeast-asian.md` — Thailand, Vietnam, Indonesia, Malaysia, Philippines, Cambodia, Laos, Singapore
- `south-asian.md` — India, Pakistan, Bangladesh, Sri Lanka, Nepal
- `mediterranean-european.md` — Italy, Spain, Portugal, Greece, southern France
- `northern-central-european.md` — France (north of Lyon), Germany, Austria, UK, Ireland, Nordic, Eastern Europe
- `latin-american.md` — Mexico, Central America, Caribbean, South America
- `middle-eastern-north-african.md` — Levant, Persia/Iran, Turkey, Arabian Peninsula, Maghreb
- `west-and-sub-saharan-african.md` — West, East, Southern Africa, Ethiopia/Eritrea

## How they evolve

Two ways:

1. **Corrections from cooking experience** — when a recipe drafted from a note drifts in a way Hunter notices and corrects, append a short observation to the relevant section. The corrections layer is the first-party knowledge that makes these notes load-bearing over time, beyond what's available from public sources.

2. **Region-specific deepening** — when a particular regional cuisine becomes a recurring focus (Sichuan, Oaxacan, Tuscan, Bahian, etc.), spin out a dedicated note. The broad continental notes stay as the top layer; specific regional notes live alongside them.

## What these are not

- Not a hard ruleset. No "must include / never include." Authority comes from being descriptive and well-sourced, not from being prescriptive.
- Not exhaustive. Each note covers what's *distinctive*; basics that recur across cuisines (rice, alliums, salt) are mentioned only where the cuisine treats them in a particular way.
- Not built into the published site. They're internal context for drafting only.
